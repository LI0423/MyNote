# YCrCb色彩空间

是一种广泛应用于图像和视频处理中的色彩空间，将颜色信息分离为亮度分量和色度分量。

## 核心组成

1. Y（亮度分量Luma）
   - 表示图像的亮度（灰度）信息
   - 包含图像的细节和结构信息
   - 类似于黑白电视信号
   - 计算公式：$Y = 0.299 * R + 0.587 * G + 0.114 * B$
2. Cr（红色色度分量Chroma Red）
   - 表示红色分量与亮度的差异
   - 范围：0-255（8位图像）
   - 计算公式：$Cr = (R - Y) * 0.713 + 128$
3. Cb（蓝色色度分量Chroma Blue）
   - 表示蓝色分量与亮度的差异
   - 范围：0-255（8位图像）
   - 计算公式：$Cb = (B - Y) * 0.564 + 128$
