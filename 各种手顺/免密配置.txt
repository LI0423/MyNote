1、添加用户  useradd 名字 -m    (四个虚拟机的名字都一样)
2、设置密码  sudo passwd 名字    在弹出来的提示窗口中设置密码即可
3、设置root用户  sudo passwd root    会弹出设置密码，设为hadoop 
4、改主机名  su root ,提示输入密码，然后  vim /etc/hostname      一个master，三个slave1,2,3
5、安装ssh  sudo apt-get install openssh-server    安装ssh服务端
           sudo apt-get install openssh-client    安装ssh客户端
6、连接本机  ssh 名字@localhost    第一次连接需要输入yes确认
7、本机免密  cd ;ssh-keygen -t rsa -P ""    进入当前用户（最好不用root）的home目录，生成本机秘钥，一路回车
           cat .ssh/id_rsa.pub >> .ssh/authorized_keys    将公钥追加到authorized_keys文件中
           chmod 600 .ssh/authorized_keys    赋予authorized_keys文件权限
8、查看ssh是否配置成功  ssh localhost
9、最终验证  exit   退出ssh当前登录   
           ssh localhost    发现不用命令也可以使用ssh登录本机即为成功
10、与其他机器连接  首先在每台虚拟机中配置  vim /etc/hosts 在文件中输入虚拟机的ip地址和主机名，例如：
           192.168.1.6 master
           192.168.1.9 slave1
           192.168.1.7 slave2
           方便向其他机器发送公钥  scp -r ~/.ssh/authorized_keys hadoop@slave1:~/.ssh/
           再进行连接测试  ssh hadoop@slave1  ,第一次连接时会输入  yes  和密码，  exit  退出连接之后，再次进行连接，就不需要了。
