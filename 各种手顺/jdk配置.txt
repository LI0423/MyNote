2、提取到桌面或者是其他文件夹中，我提取在桌面
3、配置环境变量  需要在 su - 之后进入超级管理员模式
          vim /etc/profile  进入环境变量的配置界面，可以直接在其下方插入，插入时按  i
          export JAVA_HOME=/home/hadoop/Desktop/jdk1.8.0_251
          export PATH=$PATH:$JAVA_HOME/bin
          export CLASSPATH=.:$JAVA_HOME/lib/dt.jar:$JAVA_HOME/lib/tools.jar
          插入以上这些内容之后按 Esc ，然后输入 :wq 即可保存并退出
          然后输入 source /etc/profile
5、退出管理员模式进行试验jdk是否安装成功  java -version  ,如果弹出jdk的版本信息及其他内容，即为配置成功。

在测试是否成功中遇到的问题：
   1、如果弹出sh :java :not found 这个错误，意思是说这个命令不认识，可以 cd /bin/ 查看是否有叫java的文件， 
   如果没有可以 sudo apt install busybox ,可以自动将命令加载到 /bin 文件中，安装好之后可以进行 java -version 测试。
   2、在执行 sudo 命令时，可能会报出 hadoop is not in the sudoers file 的错误，可以切换到 su 超级管理员模式，
   vim /etc/sudoers 修改文件内容，找到 root  ALL=(ALL:ALL) ALL 一行，在下面插入新的一行，内容是
   hadoop ALL=(ALL:ALL) ALL  然后 Esc 输入  ：wq! 保存并退出。（这个文件是只读的，不加“！”保存会失败）
